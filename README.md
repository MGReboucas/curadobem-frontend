# Curado Bem 🌿

Aplicativo de e-commerce feminino desenvolvido em **Flutter**, com identidade visual sofisticada em tons de verde sage e bege.

---

## Sobre o Projeto

O **Curado Bem** é uma plataforma de compras online voltada ao público feminino, com foco em uma experiência de usuário elegante, intuitiva e acolhedora. O app oferece navegação fluida e um design cuidadosamente pensado para transmitir confiança e sofisticação.

---

## Funcionalidades

- Tela de login com campos de usuário/e-mail e senha
- Cadastro de novas usuárias
- Integração com API (em desenvolvimento)
- Interface responsiva para Android e iOS

## Como Executar

**Pré-requisitos:** Flutter instalado e configurado. [Guia de instalação](https://docs.flutter.dev/get-started/install)

```bash
# Clone o repositório
git clone <url-do-repositorio>

# Instale as dependências
flutter pub get

# Execute o app
flutter run
```
